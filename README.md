# CSOPESY-Project1
The program implements a simulation of the following basic CPU scheduling algorithms:
1. First-Come First-Serve (FCFS)
2. Shortest-Job First (SJF)
3. Shortest-Remaining-Time-First (SRTF)
4. Round-Robin (RR)
